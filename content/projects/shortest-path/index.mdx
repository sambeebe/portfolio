## Shortest Path Visualization (C++/openFrameworks)
#### [code](http://github.com/sambeebe/ofxShortestPath)
Computes and visualizes the shortest path on a graph representing physical road distances in New York. containing 264,346 nodes, 733,846 arcs ([source](http://users.diag.uniroma1.it/challenge9/download.shtml)). Dijikstra's algorithm is used to compute the shortest path and I used openFrameworks to render the graph and animate the shortest path.
